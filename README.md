# Task Manager (AngularJS)

A simple and responsive Task Manager web application built using AngularJS, HTML, CSS, and Bootstrap.  
It allows users to add, edit, delete, and mark tasks as completed, with data stored in the browser using localStorage.


## Features
- Add new tasks with title and description  
- Edit and delete tasks easily  
- Mark tasks as completed using checkboxes  
- Automatically saves tasks using browser localStorage  
- Clean and responsive Bootstrap design  


## Technologies Used
- AngularJS (v1.8.2)
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Local Storage (Browser)
